# My Whisper

macOS 本地实时语音转文字工具。基于 [MLX Whisper](https://github.com/ml-explore/mlx-examples/tree/main/whisper) 在 Apple Silicon 上高效运行，所有数据完全在本地处理，不上传任何内容。

## 系统要求

- macOS 14.0+
- Apple Silicon（M1 / M2 / M3 / M4）

## 安装

1. 下载仓库中的 `My-Whisper-1.0.dmg`
2. 双击打开 DMG，将 **My Whisper** 拖入 Applications 文件夹
3. 首次打开如果提示"无法验证开发者"，前往 **系统设置 → 隐私与安全性**，点击"仍要打开"

## 首次启动

启动后窗口会显示「加载模型中...」，首次需从 Hugging Face 下载模型（约 3 GB），请保持网络畅通。下载完成后状态变为「就绪」，后续启动无需再次下载。

## 使用方法

### 录音转写

按下 `⌘⇧Space`（或点击窗口中的"开始录音"按钮）开始录音，再次按下停止。App 会自动检测语音停顿并分段转写，结果实时显示在窗口中。

### 切换语言

菜单栏 → **语言** 菜单中选择目标语言。支持：

- 中文（默认）
- English
- 日本語
- 한국어
- 自动检测

### 复制 / 清空

窗口顶部的「复制」按钮一键复制全部转写文本，「清空」按钮清除内容。也可以用 `⌘A` 全选后 `⌘C` 复制。

## 权限设置

首次使用需要在 **系统设置 → 隐私与安全性** 中授权：

| 权限 | 用途 |
|------|------|
| **辅助功能** | 全局快捷键 `⌘⇧Space` 监听 |
| **麦克风** | 语音录制 |

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘⇧Space` | 开始 / 停止录音 |
| `⌘Q` | 退出 |
| `⌘C` | 复制 |
| `⌘A` | 全选 |

## 日志

运行日志位于 `~/Library/Logs/my-whisper.log`，遇到问题时可查看排查。
